# Vimrc config

### 1. Copy .vimrc in home directory

```
~/.vimrc
```

###  2. Install Vim-plug
Run current command in home directory:
```
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```


### 3. Install plugins
Open vim and run command:
```
:PlugInstall
```
